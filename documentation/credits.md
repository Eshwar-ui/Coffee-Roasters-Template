# 📜 Credits & Resources

Espresso Elite leverages several high-quality third-party resources to provide a modern, performant experience.

## Frameworks & Libraries
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework (Loaded via CDN).
- **[Lucide Icons](https://lucide.dev/)**: Flexible, clean svg icon library.
- **[Swiper JS](https://swiperjs.com/)**: Modern touch slider for the product carousels.
- **[GSAP & ScrollTrigger](https://greensock.com/gsap/)**: State-of-the-art web animations and scroll effects.
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)**: Lightweight scroll-reveal library.

## Fonts & Imagery
- **[Google Fonts](https://fonts.google.com/)**:
    - *Playfair Display*: Elegant heading typography.
    - *Inter*: Clean, readable body text.
- **[Pexels](https://www.pexels.com/) / [Unsplash](https://unsplash.com/)**: High-resolution placeholder imagery for coffee beans, roasteries, and lifestyle shots.

## Tools Integrated
- **[Formspree](https://formspree.io/)**: Used as the default handler for contact form submissions.
- **[Google Maps](https://maps.google.com)**: Integrated via iframe for site locations.
