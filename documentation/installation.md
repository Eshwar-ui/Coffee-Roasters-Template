# 🛠 Installation Guide

Getting started with **Espresso Elite** is simple. Since it uses the Tailwind CSS CDN, you don't need to install Node.js or any build tools.

## Step-by-Step Setup

1.  **Extract the Files**: Unzip the downloaded package to your preferred location.
2.  **Open in Browser**:
    *   Double-click `index.html` to view the homepage.
    *   For a better development experience, we recommend using a local server like **"Live Server"** in VS Code to ensure all features (like navigation scripts and relative paths) work perfectly.
3.  **Edit the Code**: Use any code editor (VS Code, Sublime Text, Cursor, etc.) to modify the HTML, CSS, and JS files.
4.  **Deploy**: Upload the entire `coffee-roaster-template` folder to your web server (via FTP, Netlify, Vercel, or GitHub Pages).

## Requirements
- **Browser**: Chrome, Firefox, Safari, or Edge (latest versions recommended).
- **Internet**: Required for loading CDN assets (Tailwind, Lucide, AOS, Swiper).
