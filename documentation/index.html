Espresso Elite | Documentation

A ThemeForest-quality Specialty Coffee Roaster template.

Installation Guide

No complicated build tools required. Open pages/index.html in any modern web browser or serve via Live Server.

Customization Guide

  • Colors: We utilize Tailwind CSS in JIT mode. To update the primary/secondary colors, modify the tailwind.config object found in the <head> script tags of the HTML files.
  • Dark Mode: main.js handles the dark mode preference via localStorage. Modifying assets/css/dark-mode.css updates the inverted styles dynamically.
  • RTL System: Setting <html dir="rtl"> recalculates logic for CSS grids, swiper navigation, and icon direction via assets/css/rtl.css.

Page Explanations

  • index.html - Home 1: Product focus (capsule carousels, CTA subscriptions)
  • index-2.html - Home 2: Storytelling focus (video background, roaster background)
  • shop.html - Grid filtering system, interactive sidebars, sort tools
  • product-single.html - Subscription integration, roast charts, flavor maps
  • dashboard-admin.html - Comprehensive UI for shop management (sidebar, widgets, reports charts)
  • dashboard-user.html - Customer-facing management (active subscription status)
