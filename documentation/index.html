Introduction

Thank you for choosing Espresso Elite, a premium specialty coffee roaster HTML template. Built directly inside the browser using HTML5, advanced Vanilla CSS variables, and the Tailwind CSS CDN, this template is incredibly lightweight, blindingly fast, and requires zero build tools to modify or deploy.

Quick Start: Simply open index.html (or any file in the pages/ directory) directly in your web browser. No local server, Webpack, or npm install is required.

Styling & Custom Colors

The visual identity is controlled primarily via native CSS variables found inside assets/css/style.css. We leverage Tailwind CDN on top of these variables for spacing, typography, and utility classes in the HTML structure.

Changing the Theme Colors

Open assets/css/style.css and modify the :root block. This controls global coloring instantly.

:root {
  --primary: #6F4E37;      /* Coffee Brown - Main brand color */
  --secondary: #C4A484;    /* Light Latte - Background accents */
  --accent: #D4A373;       /* Warm Accent - Secondary buttons */
  --bg: #F5F3EF;           /* Light Paper - Main body background */
  --text: #1C1C1C;         /* Espresso - Main text color */
}

Tailwind CSS Configuration

In the <head> of every HTML page, a Tailwind configuration script automatically maps our CSS variables to Tailwind utility classes (e.g., text-primary, bg-primary). If you wish to change the primary font families, do it inside the HTML script tag: 

<script>
    tailwind.config = {
        darkMode: 'class',
        theme: {
            extend: {
                colors: { primary: '#6F4E37', secondary: '#C4A484', ... },
                fontFamily: { 
                    heading: ['Playfair Display', 'serif'], 
                    body: ['Inter', 'sans-serif'] 
                }
            }
        }
    }
</script>

Dark Mode

Dark mode is built-in natively, leveraging Tailwind's dark: variant and the assets/css/dark-mode.css stylesheet.

1. The Toggle System

The toggle functionality is powered by assets/js/main.js. Whenever a user clicks a button with the ID #dark-mode-toggle, the script adds the class .dark to the root <html> element.

2. Persistence

The JS checks the user's OS preference (prefers-color-scheme: dark) on first load. It then saves manual toggles to the browser's localStorage so the theme choice persists across all pages.

RTL Support (Right-To-Left)

Extensive care was taken to support RTL languages natively (Arabic, Hebrew, etc.).

The template includes an RTL switch on every navbar (the globe/languages icon). When clicked, it sets dir="rtl" on the HTML element. The CSS inside assets/css/rtl.css immediately flips paddings, margins, grid orders, and directional icons.

// Manual RTL testing:
<html lang="ar" dir="rtl">

Lucide Icons

We use Lucide Icons (a popular open-source SVG icon library). Instead of loading massive font files, Lucide renders crisp SVGs dynamically at runtime.

To add an icon, use an <i> tag with the data-lucide attribute that matches the icon name:

<!-- Example of rendering a coffee icon -->
<i data-lucide="coffee" class="w-6 h-6 text-primary"></i>

You can find the full list of available icon names at lucide.dev.

SEO & Accessibility

Espresso Elite is fully optimized for top-tier Google PageSpeed scores natively.

  • Images utilizing loading="lazy" on all below-the-fold media chunks.
  • WCAG 2.1 AA compliant semantic elements (<nav>, <main>, etc).
  • aria-label tags on all interactive icons and button toggles.
  • Injected JSON-LD Organization Schema in the document head for rich search results.
  • OpenGraph meta properties configured for brilliant link sharing previews.
