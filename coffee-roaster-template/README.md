# Espresso Elite - Premium Coffee Roaster HTML Template

A modern Specialty Coffee Roaster website template inspired by premium coffee capsule product designs. Built with HTML5, CSS3, Vanilla JS, and TailwindCSS (per user request).

## Features
- **Two Homepages (Capsule & Story focus)**
- **Fully Responsive** (Mobile, Tablet, Desktop, Wide)
- **Advanced Shop UI** (Grid layout, side filters, quantity selects)
- **Subscription Management UI** (Plans, user flows, delivery intervals)
- **Two Dashboards** (User & Admin with distinct layouts)
- **Dark Mode** integrated globally (localStorage synced)
- **RTL Language Support** (Fully accessible layout mirroring)
- **Interactive UI Components** powered by SwiperJS, AOS, Lucide Icons

## Technologies
- **Styling:** TailwindCSS CDN configured dynamically. Custom CSS in `assets/css/` overriding scrollbars, animations, components.
- **Icons:** Lucide-Icons (Vanilla JS API).
- **Animations:** AOS (Animate On Scroll) & Custom transitions in CSS.
- **Carousel:** SwiperJS (v10).
- **Fonts:** Playfair Display (Headings), Inter (Body).

## Map Integrations
Refer to `product-single.html` to see the Coffee Origins Map placeholder implementation. 

## Structure
- `pages/` - Contains all HTML routing endpoints (`index.html` is Home 1).
- `assets/` - Contains minified images (webp), javascript controllers, and CSS layers.
- `documentation/` - View the documentation index.html for guides.

## Running Locally
This template relies purely on static front-end technologies.
Simply open `pages/index.html` in your favorite modern browser, or run a local server:
```
python -m http.server 8000
```
